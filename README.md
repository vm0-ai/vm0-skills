# vm0-skills

A collection of reusable [Agent Skills](https://agentskills.io) for AI agents.

## Skills

| Skill | Description |
|-------|-------------|
| [rss-fetch](./rss-fetch) | Fetch and parse RSS feeds from provided URLs |
| [fal-image](./fal-image) | Generate images using fal.ai API |
| [devto-publish](./devto-publish) | Publish articles to Dev.to |
| [github](./github) | GitHub operations via gh CLI |

## Usage

Each skill folder contains:
- `SKILL.md` - Skill definition and instructions
- `scripts/` - Executable scripts (optional)

Skills follow the [Agent Skills specification](https://agentskills.io/specification).
